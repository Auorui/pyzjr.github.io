# pyzjr.videos

本文作者：Auorui(夏天是冰红茶) (opens new window)

本站地址：https://pyzjr.github.io (opens new window)

# Parameters

# pyzjr.videos.VideoCap()

class，自定义的视频捕捉类。

  • CapInit(self, mode=0, w=640, h=480, l=150)

初始化镜头参数，

参数

mode 默认调用本地摄像头

w 表示窗口宽度

h 表示窗口高度

l表示亮度

  • read(self, flip=None)

镜头读取函数

参数

flip: -1: 水平垂直翻转,

            0: 垂直翻转,

            1: 水平翻转

  • free(self)

摄像头释放函数

# pyzjr.videos.FPS()

class，在窗口左上角打印帧率。

  • update(self, img=None, pos=(20, 50), color=(255, 0, 0), scale=3, thickness=3)

更新帧速率

参数

img 显示的图像,如果只需要fps值,则可以留空。

pos 图像上FPS上的位置。

color 显示的FPS值的颜色

scale 显示的FPS值的比例

thickness 显示的FPS值的厚度

# pyzjr.videos.Timer()

class，封装了time库的一个类，常用于计算代码运行时间

  • start(self)

开始计时

  • stop(self)

停止计时

  • avg(self)

返回平均时间

  • total(self)

返回总时间

  • cumsum(self)

返回累计时间

# pyzjr.videos.Mp4toGif(mp4, name='result.gif', fps=10, start=None, end=None)

将视频转为Gif图

参数

mp4 需要转换的视频

name 保存的Gif图路径名

fps 帧率转换为每帧之间的延迟时间（毫秒）

start 一般是全部转换，所以为None

end 一般是全部转换，所以为None

返回

  保存好的Gif图

# 示例

import pyzjr.Z as Z
fpsReader = FPS()
Vcap = VideoCap()
Vcap.CapInit(mode=0)
while True:
    img = Vcap.read()
    fps, img = fpsReader.update(img)
    cv2.imshow("Image", img)
    k = cv2.waitKey(1)
    if k == Z.Esc:
        break
