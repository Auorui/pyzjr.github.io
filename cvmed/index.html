# Parameters

# pyzjr.mediapipe.HandTrack.Hands()

class，基于mediapipe的手势追踪

  • __init__(self,NumHands=2,minDetconfid=0.5,minTrackconfid=0.5,StaticMode=False)

初始化构造函数

参数

NumHands 识别手的数量

minDetconfid 识别置信度

minTrackconfid 追踪置信度

StaticMode 静态检测模式

  • findHands(self, img, draw=True, flipType=True, showcor=False)

在BGR图像中查找手,内部会强制转换为RGB

参数

img 图像

draw 在图像上绘制输出的标志

返回

  带或不带标志的图像

  • gesture(self, myHand)

手势形状,分别考虑左手和右手

参数

myHand 单个手的信息字典

返回

  以列表的形式展现手指状态

  • fingerSpace(self, p1, p2, img=None, color=Z.green, radius = 15, line_thickness = 2)

计算两个指定点之间的距离，并在图像上绘制相关信息（可选）。

参数

p1 Point1，(x1, y1)

p2 Point2，(x2, y2)

img 要绘制在上面的图像（可选）

color 绘制的颜色

radius 绘制圆的半径

line_thickness 线条厚度

返回

  如果提供了图像，则返回信息和已绘制的图像 如果没有提供图像，则返回信息

示例

from pyzjr.videos import VideoCap
Vap=VideoCap()
Vap.CapInit()
detector = Hands()
while True:
    img = Vap.read()
    hands, img = detector.findHands(img)
    if hands:
        hand1 = hands[0]
        lmList1 = hand1["lmList"]
        fingers = detector.gesture(hand1)
        if len(hands) == 2:
            hand2 = hands[1]
            lmList2 = hand2["lmList"]
            info, img = detector.fingerSpace(lmList1[8], lmList2[8], img)
    cv.imshow("Image", img)
    if cv.waitKey(1) == Z.Esc:
        break
