# pyzjr.Graphmath

本文作者：Auorui(夏天是冰红茶) (opens new window)

本站地址：https://pyzjr.github.io (opens new window)

# Parameters

# pyzjr.Graphmath.EuclideanDis(pts1, pts2)

欧式距离与中心点

参数

pts1 第一个目标点

pts1 第二个目标点

返回

  返回两点之间欧式距离以及二者的中心点

# pyzjr.Graphmath.retain(val,t=2)

精准保留小数,默认2位

参数

val 要进行处理的值

返回

  返回小数点后两位的浮点数

# pyzjr.Graphmath.normal(x, mu, sigma)

正态分布（高斯分布）概率密度函数

参数

x 要进行处理的值

mu 均值

sigma 标准差

返回

  正态分布后的结果

# pyzjr.Graphmath.Gaussian2D(x, y, sigma)

计算二维高斯函数的数值

# pyzjr.Graphmath.Advancedlist(imglist,mode='T',reverse=False)

对列表的高级排列

参数

imglist 要进行处理的值

mode “T”:平铺; “F”:首元素; “X”:按照x排序,默认是从小到大; “Y”:按照y排序,默认是从小到大;

reverse 是否进行反转，只对某些模式有用

返回

  返回重新排列后的列表

# pyzjr.Graphmath.angle_to_2pi_range(angle)

将角度映射到0到2π

# pyzjr.Graphmath.degree_to_radians(angle_degrees)

度数转弧度

# pyzjr.Graphmath.rand(a=0., b=1.)

生成在指定范围内的随机浮点数,进行缩放和偏移来映射到[a, b)的范围

参数

a 下界

b 上界

返回

  随机浮点数

# pyzjr.Graphmath.NumericalMeter()

class，计算并存储平均值和当前值

  • reset(self)

重置函数，将记录的值归0

  • update(self, val, n=1)

数值更新，记录每一步传入数据后的记录

参数

val 需要记录的值

n 次数，每隔多少进行一次记录，大多数为1
