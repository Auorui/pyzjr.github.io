# Parameters

# pyzjr.getFilePath(filedir,format='png')

比如：

- root_directory
- folder1
    - file1.png
    - file2.txt
- folder2
    - file3.png
    - file4.jpg
- file5.png
- file6.txt

getFilePath(root_directory)得到的是

['root_directory/folder1/file1.png', 'root_directory/folder2/file3.png', 'root_directory/file5.png']

参数

filedir 表示目标路径

format 指定的后缀文件，默认是png

返回

  返回的是一个列表，包含目标路径下所有符合后缀的路径。

# pyzjr.getPhotopath(paths,cd=False,debug=True)

获取指定文件夹路径下的所有文件的路径，建议最好是使用绝对路径

参数

paths 文件夹路径

cd 添加当前运行的路径名,使用了相对路径才能用的

debug 开启打印文件名错误的名字

返回

  返回两个参数，第一个是只有图片路径，第二个是指定路径下所有文件路径的列表.

# pyzjr.mkdir(folder_path,debug=False)

确保文件存在，如果不存在就会创建，并且会返回这个文件路径

参数

folder_path 要检查的文件夹路径

debug 打印文件的当前信息（是否存在，是否创建），可以直接False

返回

  返回输入文件的路径。

# pyzjr.ImageAttribute(image)

打印出一些图片文件的信息

参数

image 支持cv2.imread读取的图像或图像路径

返回

  返回图片的一些基础信息，比如基础名，格式，字节，类型等。

# pyzjr.logdir(dir="logs", format=True, prefix="", suffix="")

一个日志生成器

参数

dir 默认名为“logs”

format 提供了两种日志时间类型str

prefix 在logs与日志时间类型str之间添加字符

suffix 在日志时间类型str之后添加字符

返回

  示例1：返回logs\abc\Monday_11_December_2023_22h_25m_15s\def, (format=True,prefix="abc",suffix="def")

  示例2：返回logs\abc\2023_12_11_22_26_58\def, (format=False,prefix="abc",suffix="def")

# pyzjr.get_levels_paths(path)

获取各级的文件路径

比如：

“D:\PythonProject\Torchproject\Lasercenterline\tst\ssa”

获取的就是下面的一个列表：

[WindowsPath('D:/'),

WindowsPath('D:/PythonProject'),

WindowsPath('D:/PythonProject/Torchproject'),

WindowsPath('D:/PythonProject/Torchproject/Lasercenterline\tst'),

WindowsPath('D:/PythonProject/Torchproject/Lasercenterline\tst/ssa')]

参数

path 输入目标路径

返回

  两个参数，第一个是包含了各级目录的列表，第二个参数是最大的索引值

# pyzjr.Data2csv(header, value, log_dir, savefile_name)

将数据转为csv表格

参数

header 列的标题

value 对应列的值

log_dir 文件夹路径

savefile_name 文件名（包括路径）

返回

  将数据写入了csv表格当中

# pyzjr.Microsoft2PDF()

class，其构造函数分别用于将ppt,word,excel转换为PDF.

参数

__init__里面只用传入filePath，即目标路径，可以不填，这样默认就是当前路径下。

返回

  调用WEP2Pdf构造函数，其可以识别该路径下word,excel,ppt将其转为对应的pdf.

# pyzjr.is_url(filename)

如果字符串是http或ftp路径，则返回True。

参数

filename 要检查的字符串

返回

  布尔值

# pyzjr.is_ascii(s)

检查字符串是否仅由ASCII字符组成。

参数

s 要检查的字符串

返回

  布尔值

# pyzjr.file_age(path, detail=False)

获取路径文件自上次文件更新以来的天数。

参数

path 要检查的路径

detail 是否要将秒数也算上，这里默认False

返回

  返回自上次文件更新以来的天数。

# pyzjr.file_date(path=file)

返回可读的文件修改日期,i.e:'2021-3-26'。

参数

path 要检查的路径

返回

  返回可读的文件修改日期

# pyzjr.file_size(path)

获取文件/目录的大小（以MB为单位）。

参数

path 要检查的路径

返回

  返回文件/目录大小（MB）。
