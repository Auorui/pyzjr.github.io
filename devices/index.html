本文作者：Auorui(夏天是冰红茶) (opens new window)

本站地址：https://pyzjr.github.io (opens new window)

# Parameters

# pyzjr.cpu()

返回CPU设备。

返回

  torch.device: The CPU device.

# pyzjr.gpu(i=0,cuda=True)

如果cuda为True并且gpu可用,则返回gpu(i),否则返回cpu()。

参数

i 一般来说用不到第几块gpu,参数可以保持默认为0， cuda 表示是否使用cuda，如果环境中没有cuda，改为True也没用

返回

  返回gpu设备，如果不是则返回cpu设备

# pyzjr.allgpu()

如果哪天会调用这个函数，说明我十分的富有

返回

  返回所有的gpu设备（即使用多卡）

# pyzjr.num_workers(batch_size)

确定用于数据加载的并行工作进程的数量

参数

batch_size 表示批处理

返回

  返回的是一个整数，表示用于数据加载的最终并行工作进程的数量。
